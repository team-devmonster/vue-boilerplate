## Cypress

```ssh
// bash shell (or Powershell)
$ setx CYPRESS_CACHE_FOLDER "C:\CypressCache"

$ export CYPRESS_CACHE_FOLDER="C:/CypressCache"

$ npx cypress install

$ npx cypress open
```
